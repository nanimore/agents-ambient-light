# 项目总结

## 🎯 项目概述

**Agents Ambient Light** 是一个为 AI 编程助手（Claude Code、Codex 等）设计的屏幕氛围灯通知系统。通过在屏幕边缘显示平滑渐变的彩色光效，为用户提供优雅的视觉反馈。

## 📊 项目规模

- **代码量**：约 400 行 Python + 150 行 Shell/PowerShell
- **文件数量**：12 个文件
- **项目大小**：约 34 KB（不含依赖）
- **开发时长**：1 天（含技术方案迭代）

## ✨ 核心技术亮点

### 1. 完美的透明渐变
经过 4 次技术方案迭代，最终使用 **PyQt5 + QLinearGradient** 实现真正的 alpha 透明渐变，无硬边、无黑边、无白底。

### 2. 跨平台支持
- Windows：PowerShell 集成
- macOS/Linux：Bash 集成
- 统一的 YAML 配置格式

### 3. 轻量设计
- 启动时间 < 1 秒
- 内存占用 < 50 MB
- 自动退出，无后台常驻

## 🎨 功能特性

### 基础功能
✅ 屏幕边框氛围灯（四周/顶底）
✅ 7 种预设颜色 + 自定义颜色
✅ 3 种动画（呼吸灯/闪烁/静态）
✅ 可调宽度（20-80 像素）
✅ 可调持续时间（1-60 秒或持续显示）

### 高级功能
✅ 平滑的边缘到中心 alpha 渐变
✅ 事件颜色映射（stop/input/task_complete）
✅ 三种通知模式（氛围灯/声音/闪烁）
✅ YAML 配置热加载
✅ 自动进程清理
