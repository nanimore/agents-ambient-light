# 更新日志

本项目的所有重要变更都将记录在此文件中。

## [1.0.0] - 2026-06-14

### 新增功能
- ✨ 屏幕氛围灯，带平滑渐变效果
- 🎨 支持 7 种预设颜色 + 自定义十六进制颜色
- 💫 三种动画模式：呼吸灯、闪烁、静态
- 🎚️ YAML 配置文件，轻松自定义
- 🔇 灵活的通知模式（氛围灯/声音/任务栏）
- 🖥️ 跨平台支持（Windows、macOS、Linux）
- 📝 完整的 README 和安装脚本
- ⚡ 定时自动退出，进程清理完善

### 技术实现
- 使用 PyQt5 实现真正的 alpha 透明支持
- 使用 QLinearGradient 实现平滑的边缘到中心渐变
- 轻量级：约 200 行 Python 代码，启动时间 <1 秒
- 除 PyQt5 外无其他硬性依赖

### 项目演进历程
为了实现完美的透明效果，本项目经历了多次迭代：

1. **v0.1 - Tkinter + 矩形分层**
   - 问题：无真正 alpha 支持，仅支持亮度渐变
   - 结果：渐变到黑色（#000000）时出现深色边缘

2. **v0.2 - Tkinter + Numpy + PIL RGBA**
   - 问题：Tkinter 不原生支持 RGBA 图像
   - 结果：屏幕中心显示白色/灰色背景

3. **v0.3 - Tkinter + transparentcolor 技巧**
   - 问题：只有纯洋红色（#FF00FF）会变透明
   - 结果：渐变与透明区域交界处出现硬边

4. **v1.0 - PyQt5 + QLinearGradient** ✅
   - 解决方案：Qt 原生支持 alpha 通道
   - 结果：完美的平滑渐变，从边缘到透明中心
